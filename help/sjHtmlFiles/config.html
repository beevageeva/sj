Config files

Config files are XML files with config as root element.
elementattrcontent descr
config-virtualAddressNBits , numberProcessesNBits ,diskATU, ,pageTable , mainMemoryConfig , cacheConfig , cacheConfig , cacheConfig
virtualAddressNBits - {the number of virt pages for each process. default :32}
numberProcessesNBits - {the p2 of max number of processes. default :0}
diskATU - {disk access time units. default :1}
pageTabledirecttlbConfig , (offsetLengths , searchMethod)||(hashAnchorSizeNBits) { If attribute direct has value true it will look for elements offsetLengths , searchMethod (this means that is direct mapped, set the page table as direct mapped); otherwise it will look for elem hashAnchorSizeNBits (sets the page table as inverse mapped)}
offsetLengths-length ,.. length
length-{the length in bits of an offset for a level.they are taken in the order they appear in the list}
searchMethod-{top down or bottom up.if the string is equal to topDown set meth to top down ; otherwise to bottom up}
hashAnchorSizeNBits-{p2 of the hash anchor size in the case of the inverse mapped page table}
pageAgingConfig-pageAgingIncrease ,memRefToBeRun
pageAgingIncrease-{the age units incresed for an entry when it's referenced}
memRefToBeRun-{the number of mem references before page aging thread is run . To disable put this field -1. By default is disabled}
memAllocConfig-minPFF , maxPFF , evNodesToRun
minPFF-min PFF
maxPFF-maxPFF
evNodesToRun-{the number of evicted pages from the main memory before this thread is run.To have a local alocation policy and also disable this thread put this in -1.Default is local}
tlbConfig , mainMemoryConfig , cacheConfig - numberEntriesNBits , blockSizeNBits , evictionPolicy , busSize numberSetsNBits , accessTimeUnits , dataInstrSeparated
numberEntriesNBits-{p2 of the number of entries in the cache}
blockSizeNBits-{p2 of the block size of the cache}
evictionPolicy-{the eviction policy.if string equals to random set random EP , if string equals to fifo set fifo EP , if string equals to lru set lru EP , if string equals to lfu set lfu EP , if string equals to nru set nru EP , if string equals to nfu set nfu EP}
busSize-{p2 of the number of entries in the cache.}
accessTimeUnits-{access time units .default is 1}
numberSetsNBits-{p2 of the number of sets in the cache(direct , fully assoc , set assoc are part cases).}
dataInstrSeparated-{if value is true data and instr are sep, otherwise they aren't(only a table is created)}
hitWritePolicy - {the cache hit write policy: may be write through or write back.if the string is equal to writeThrough it will be set to write through , otherwise to write back.Default writeBack}
missWritePolicy - {the cache miss write policy: may be write allocate or no write allocate.if the string is equal to writeAllocate it will be set to write allocate , otherwise to no write allocate}.Default noWriteAllocate
All the settings that don't appear as input fields in config panels are ignored, thus busSize is ignored for tlb , numberSetsNBits ignored for mainMem , dataInstrSeparated ignored for other caches than C1 and tlb, for all caches(if enabled : C1 , C2 , C3) is set the same block size and in the form will appear the blockSize of C1(C1 corresp to first cacheConfig element encountered, C2 to the second , C3 to the third) If not set in this config file , the field will take def values.
