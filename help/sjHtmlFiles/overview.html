Start

1)configure(open a configuration file from Config Menu)
2)pass over the screens of configuration(local validations in the screen are made when pressing next or prev ; global validations : that is validations that depend on fields that are in differnt screens are made when pressing OK). OK button appears enabled only after passing through all configuration screens in order that all local validations are made.
3)press Start Menu Item from Actions Menu (the other tabs will appear)
4)load a trace file from Trace menu

Reconfig Menu Item from Config Menu removes all other tabs except conf and also removes all the trace files loaded
